## 📋 프로젝트 소개

사용자의 출석 현황을 한눈에 파악할 수 있으며, 개인별 통계도 제공합니다.

## ✨ 주요 기능

- **출석 체크**: 사용자는 자신의 출석 및 퇴근 시간을 기록할 수 있습니다.
